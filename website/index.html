How It Works

Three steps to better code

Paste a repo, let AI agents review, get actionable findings with suggested fixes. No configuration needed.

01
📋
Point at your code
Paste a GitHub URL, run the CLI, or trigger from a PR. CodeAudit gathers context — diffs, imports, dependencies, and your team's review rules.
02
🧠
5 agents review in parallel
Security, Architecture, Performance, Functional, and Maintainability agents each independently analyse your code. Then a Judge agent deduplicates and ranks findings.

Receive a ranked report with severity, confidence scores, code snippets, and suggested fixes. Export as Markdown, SARIF, or inline PR comments.

Powerful CLI

Run code audits from anywhere. Three modes to choose from — quick (2 min), deep (8-15 min, all 5 agents), and security-focused. Output to terminal, Markdown, or SARIF.

  • code-audit review --mode deep
  • code-audit review --mode security
  • code-audit init (generate REVIEW.md)
  • Markdown + SARIF report output

Secrets Detection

Deterministic regex scan runs before LLM agents at zero cost. Catches hardcoded API keys, tokens, passwords, and private keys with 30+ patterns.

  • AWS keys, GitHub tokens, Slack tokens
  • Private keys (RSA, EC, DSA, OPENSSH)
  • High-entropy string detection
  • Smart allowlist (skips tests, examples)

Dependency Audit (SCA)

Automatically detects your package manager and checks for known CVEs using free tools — npm audit, pip-audit, osv-scanner, cargo-audit. Zero cost.

  • npm / pip / go / cargo supported
  • CVE severity mapping
  • Graceful degradation (missing tools)
  • osv-scanner as universal fallback
